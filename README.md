# Personal Portfolio Website

A clean, responsive personal portfolio for **Prakash Kumar Yadav**. The site highlights services, featured projects, skills, and provides a contact form for project inquiries.

## Pages
- `index.html` — Home page with hero intro, services, featured projects, skills, and CTA
- `about.html` — Background, approach, milestones, and strengths
- `contact.html` — Contact details and inquiry form

## Features
- Sticky navigation with active states and CTA button
- Responsive layout using CSS Grid and Flexbox
- Card-based sections for easy scanning
- Contact form with client-side validation and status messages
- Consistent visual system with CSS variables and custom typography

## Tech Stack
- **HTML5** for semantic structure
- **CSS3** for styling, layout, and responsiveness
- **JavaScript** for form validation and interaction

## Project Structure
- `index.html`
- `about.html`
- `contact.html`
- `css/style.css`
- `js/script.js`

## How to Run
1. Open `index.html` in a browser.
2. Navigate using the top menu links.

Tip: For best results, serve the folder with a local server (optional) to avoid any browser restrictions.

## Notes
- The contact form is **client-side only**. To send real emails, connect the form to a backend or a form service.
